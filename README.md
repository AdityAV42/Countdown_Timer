# Countdown Timer

A JavaScript frontend project to create a countdown timer.

## Functionalities  
- Users can reset the timer.  
- Option to select the end date and time.  
- Beep sound when the timer expires.  

## Future Enhancements  
- Implement notifications and emails for:  
  - **5 minutes remaining**  
  - **1 minute remaining**  
  - **Timer expired**
